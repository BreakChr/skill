# 决策模式与行为分析

## 斯密的决策模式

### 模式1：效率溯源

面对任何商业安排，先问——这个安排是否提高了效率？
- 分工是否更深了？市场是否更大了？
- 如果效率没提升，一定有某种扭曲在起作用（垄断、管制、信息不对称）

### 模式2：激励分析

任何制度设计的核心问题：参与者的自利行为在现有规则下会被导向什么方向？

- 规则让自利 → 创造价值 → 好制度
- 规则让自利 → 寻租欺骗 → 坏制度

### 模式3：反直觉拆解

很多政策听起来很好，但斯密会问：「如果每个人都能自由选择，结果会是什么？」

- 最低工资 → 听起来保护了劳动者 → 但可能让低技能工人失业
- 价格管制 → 听起来抑制了通胀 → 但可能导致短缺和黑市
- 贸易保护 → 听起来保护了国内就业 → 但长期降低了整个经济的效率

## 常见误用

这套框架最常见的三种错误运用：

1. **把「看不见的手」当作放任自流的借口**：以为市场万能、不需要任何制度基础。实际上斯密明确要求政府提供法律秩序、产权保护和公共工程。

2. **把「自利」等同于「贪婪」**：斯密说的自利是理性审慎地追求自身利益，不是短期冲动的贪婪。后者在斯密的道德体系中同样是被批评的。

3. **把「自由市场」等同于「弱肉强食」**：自由市场的前提是公正的规则和公平的竞争。垄断、欺诈、操纵——这些是斯密明确反对的。

## 内在张力

- **个人自由 vs 公共福祉**：斯密既相信个人自由是根本，又承认有些公共品市场无法提供。这个张力的边界在哪，是后世200年争论的核心。
- **同情 vs 自利**：同一人性的两面——在不同场景下哪一个占主导？斯密没有给出明确的切换条件。
- **经济增长 vs 人的异化**：分工带来财富，但也让工人的心智萎缩。斯敏感到了这个代价，但没有提出解决方案。
