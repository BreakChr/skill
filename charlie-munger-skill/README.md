# charlie-munger-skill

> *"You've got to have models in your head. And you've got to array your experience—both vicarious and direct—on this latticework of models."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**查理·芒格的多元思维框架。用跨学科的思维格栅模型拆解投资决策、商业判断和人生选择。**

基于《穷查理宝典》、25种人类误判心理学、南加州大学/哈佛"普世智慧"演讲、伯克希尔股东会问答，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/charlie-munger-skill
```

或复制到 `~/.config/opencode/skills/charlie-munger-skill/`。

## 使用示例

### 问：想投资一只热门AI股票，但怕追高

回答会先从能力圈角度分析——你对AI的理解是否足够让你判断它的长期价值？再用逆向思维列出追高会失败的原因（估值泡沫、技术路线不确定性），然后用双轨分析同时做理性和心理检查。

### 问：我总在股票下跌时舍不得卖，越亏越多

回答会用人类误判心理学的25种倾向来诊断——你被"损失厌恶"（厌恶确定的亏损）和"承诺与一致性倾向"（不愿承认之前买错了）双重绑架了。然后给出用检查清单来对抗偏误的具体方法。

### 问：想创业但不知道做什么方向

回答会用多元思维格栅来交叉验证你的想法——经济学的供需结构是什么？生物学的竞争环境如何？心理学的用户激励怎么设计？不需要全部完美，至少三个模型指向同一个方向才值得投入。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"投资大师语录合集"不同，这套框架的出发点是：

1. **从底层思维逻辑出发**：不教你怎么买股票，教你"如何用跨学科模型独立思考判断"，你拿到这个工具箱，就能分析任何问题。

2. **可操作的决策工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
