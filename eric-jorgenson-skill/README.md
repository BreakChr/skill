# eric-jorgenson-skill

> *"Wealth is assets that earn while you sleep. Happiness is a skill you can practice."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**基于纳瓦尔（Naval Ravikant）财富与幸福哲学的思维框架。用专长×杠杆×执行力的底层逻辑拆解职业路径、商业模式和人生决策。**

基于《纳瓦尔宝典》（The Almanack of Naval Ravikant）、纳瓦尔播客、推特风暴、Evergreen博客，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/eric-jorgenson-skill
```

或复制到 `~/.config/opencode/skills/eric-jorgenson-skill/`。

---

## 使用示例

### 问：我996做了5年，感觉没有成长

回答会用"杠杆三件套"分析你当前的收入是否在出租时间，再用"专长识别器"帮你找到你的特定知识，最后用"复利识别镜"规划一条有累积效应的路径。

### 问：想辞职做自媒体，靠谱吗？

回答会从"财富定义器"出发，帮你区分做自媒体是换了一种方式出租时间（接广告也是出卖注意力），还是建立了真正的资产（内容资产持续带来被动收入）。

### 问：赚到多少钱才能幸福？

回答会反问"你确定你问的是钱的问题吗？"然后引向"幸福练习法"，帮你区分欲望驱动和内在驱动，指出幸福是一种技能而不是一个数字。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

1. **从第一性原理出发**：不从具体建议出发，而是教你"怎么用基本学科原理重建自己的认知体系"——你拿到这个判断标准，自己就能分析任何局面。

2. **可操作的判断工具**：每个思维模型附带具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
