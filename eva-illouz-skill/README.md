# eva-illouz-skill

> *"Emotional capitalism is a culture in which emotional and economic discourses and practices mutually shape each other."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**伊娃·易洛思的情感社会学思维框架。用情感资本主义的底层逻辑拆解亲密关系、消费文化和社会心理结构。**

基于《冷亲密：情感资本主义的形成》《为什么爱会伤人》《消费浪漫乌托邦》《奥普拉·温弗瑞与苦难的魅力》等著作、30+学术讲座和批评者分析，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/eva-illouz-skill
```

或复制到 `~/.config/opencode/skills/eva-illouz-skill/`。

## 使用示例

### 问：为什么我在约会App上总是找不到合适的伴侣

回答会用"情感资本主义检测器"和"婚姻市场与性场域"框架拆解——这不是你个人条件或心理的问题，而是选择池膨胀、评估标准市场化、性别不平等在数字场域中的结构性后果。然后给出"归因拆分"的方法，区分个人因素和结构因素。

### 问：为什么现在的人谈恋爱越来越像在做"商业谈判"

回答会用"冷亲密诊断器"分析亲密关系的冷却过程——情感被市场话语体系重新编码，情感投入被计算为"投入产出比"，关系动力被"有效沟通"技术取代。然后给出重新叙事的方法。

### 问：社交媒体上大家都在展示痛苦，这是真实的吗

回答会用"痛苦的文化商品化"模型拆解——痛苦经历了从私密忍受→公共叙事→符号资本的转换。展示痛苦既是真实的，也是被资本的格式（注意力经济）重新编码的。然后引导用户区分"痛苦的体验"和"痛苦叙事的格式"。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数学术概念科普不同，这套框架的出发点是：

1. **从底层逻辑出发**：不教你怎么谈好恋爱或管理情绪，教你"情感和经济逻辑如何在你的日常体验中纠缠"，你拿到这个判断标准，自己就能分析任何情感/文化现象。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
