# 核心原理详解

## 量化投资的第一性原理

西蒙斯体系最底层的判断：**市场不是随机游走的。** 在看似混沌的价格波动中，存在可识别的统计规律。

- 大部分波动是噪声 → 不能被预测
- 一小部分波动是信号 → 可以被捕捉
- 信号极其微弱 → 需要数学工具放大
- 信号短暂存在 → 需要高频执行捕捉

传统投资者试图理解"为什么"（为什么涨、为什么跌）。西蒙斯只关心"是什么"——数据中是否出现可复制的模式，不管原因是什么。这个区别是一切的前提。

## 统计显著性 vs 实际显著性

量化交易中最常见的错误：混淆统计显著和实际显著。

- **统计显著**：p值小于0.05，说明模式不太可能是偶然
- **实际显著**：扣掉交易成本、滑点、冲击成本后仍然盈利

大奖章基金的成功不是因为它找到了统计显著的模式——很多人都能找到。它成功的原因是**它在扣除高额费用后仍然有实际显著的正收益**。交易成本吃掉了很多量化策略的利润，但吃掉不了大奖章的。

## 过拟合的陷阱

这是量化研究者最常犯的错误，也是西蒙斯团队花最多精力避免的问题：

- 在历史数据上拟合出完美的策略 → 看起来年化50%
- 放到未来数据上测试 → 变成亏损

西蒙斯的应对方法：
1. 尽量简化模型，参数越多越容易过拟合
2. 使用样本外数据严格验证
3. 使用跨市场/跨周期数据测试鲁棒性
4. 如果回测结果"好得不真实"，那就不是真的

## 关于竞争与市场效率

一个常被忽略的事实：量化策略的收益会随着参与者的增多而衰减。

大奖章基金封闭运营、不接受外部投资者的一个重要原因——如果太多资金追逐同样的信号，这些信号会迅速消失。西蒙斯的策略优势建立在"别人不知道我在做什么"的基础上。

这对个人投资者的启示：公开分享的量化策略几乎一定已经失效。真正有效的策略不会被分享。
