# jesse-livermore-skill

> *"Markets are never wrong — opinions often are."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**杰西·利弗莫尔的交易与投机思维框架。用趋势跟踪、关键点策略和资金管理拆解市场行为、交易决策和风险控制。**

基于《股票大作手回忆录》《How to Trade in Stocks》、21条交易规则和其四起四落的传奇生平，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/jesse-livermore-skill
```

或复制到 `~/.config/opencode/skills/jesse-livermore-skill/`。

## 使用示例

### 问：最近跌了很多，要不要抄底？

回答不会直接说"买"或"不买"，而是用"最小阻力路径"帮你判断——当前趋势是向下还是已经反转？关键点出现了吗？如果最小阻力方向还是向下，你的"抄底"就是在逆势交易。

### 问：盈利了但不敢持仓，总想卖

回答会用"心理诊断仪"帮你识别恐惧在作祟，然后用"金字塔加仓器"的逻辑提醒你：如果趋势还在，大钱是坐着赚来的。当害怕利润消失的时候，答案是移动止损，不是卖出。

### 问：要不要听朋友推荐的股票

回答会直接引用利弗莫尔的规则："如果有人把稳赚不赔的消息塞给你，那他一定不缺钱。"然后从四步清单帮你建立自己的决策系统——不看消息，看趋势和关键点。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

1. **从底层逻辑出发**：不教你怎么选股买基金，教你"趋势、关键点、资金管理"——你拿到这些判断标准，能自己分析任何市场。

2. **可操作的判断工具**：每个思维模型附带具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注利弗莫尔四起四落的警示和自杀结局的反思，避免盲目复制其高风险策略。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
