# jintao-skill

> *"人生发财靠康波。我们每个人的财富积累，一定不要以为是你多有本事，财富积累完全来源于经济周期运动的时间给你的机会。"*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**周金涛的经济周期思维框架。用康波周期（康德拉季耶夫周期）的底层逻辑拆解宏观经济、资产配置和人生财富规划。**

基于《涛动周期论》《人生财富靠康波》系列著作、宏观对冲研报和四周期嵌套模型，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/jintao-skill
```

或复制到 `~/.config/opencode/skills/jintao-skill/`。

## 使用示例

### 问：现在适合买房吗？

回答会先定位你处在康波的什么阶段、地产周期处于什么位置，然后用四周期嵌套分析当前房市是反弹还是反转，最后给出基于周期位置的判断——"2014年之后不买房"这条纪律现在是否仍然适用。

### 问：A股现在能不能抄底？

回答会用周期共振研判：目前长波（康波萧条末期）、中波（朱格拉周期底部）、短波（库存周期位置）各处于什么方向，三周期是否形成共振。如果共振向上则机会来临，如果只是短周期反弹则提醒风险。

### 问：AI时代怎么投资？

回答会从康波周期的技术创新驱动力出发，分析AI是否是第六波康波的主导技术，然后判断当前处于新技术周期的哪个阶段——复苏期还是繁荣初期，再给出对应的资产配置建议。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"经济预测合集"不同，这套框架的出发点是：

1. **从周期底层逻辑出发**：不教你怎么选股、何时买卖，教你"如何判断当前处在周期的什么位置"，你拿到这个定位标准，自己就能分析任何资产类别。

2. **可操作的研判工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
