# john-kenneth-galbraith-skill

> *"The affluent society is one in which the wants of the individual are not innate but contrived by the process by which they are satisfied."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**约翰·肯尼思·加尔布雷思的制度经济学思维框架。用抗衡力量、传统智慧和制度结构的底层逻辑拆解市场权力、公共政策和企业行为。**

基于《富裕社会》《新工业国》《美国资本主义》《1929大崩溃》等经典著作，提炼为可运行的思维框架。

## 一键安装

```bash
npx skills add <你的GitHub用户名>/john-kenneth-galbraith-skill
```

或复制到 `~/.config/opencode/skills/john-kenneth-galbraith-skill/`。

## 使用示例

### 问：大厂垄断，小企业还有活路吗？

回答会用"抗衡力量"框架分析：大厂虽强，但与其说你在和它们竞争消费者，不如说你在成为某种制衡力量——就像沃尔玛制衡宝洁一样。然后帮你定位你自己可以成为哪一类抗衡角色。

### 问：人人都在焦虑、报课、消费升级，这是真实需求吗？

回答会启动"依赖效应"诊断：你的焦虑是怎么来的？是生活真的有问题，还是某种生产机制在制造"你不够好"的感觉？然后帮你区分真实需求和被生产的需求。

### 问：政府该不该大力投资基建？

回答会用"社会平衡"框架：不只看"财政赤字"，而是看你所处的社会在私人消费和公共服务之间的错位程度。如果学校快塌了而手机每年换，问题不是钱的问题，是优先级的问题。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"经济学常识科普"不同，这套框架的出发点是：

1. **从制度出发**：不教你怎么"正确"理解供需曲线，教你"市场权力在谁手里，谁在塑造谁的需求"——拿到这个分析工具，你自己就能拆解任何经济现象。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
