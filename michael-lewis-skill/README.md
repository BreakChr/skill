# michael-lewis-skill

> *"The world is not the way the people in charge say it is."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**迈克尔·刘易斯的叙事非虚构与行为金融分析框架。用反直觉洞察、信息不对称和系统失灵的底层逻辑拆解市场机制、组织行为和社会结构。**

基于《说谎者的扑克牌》《点球成金》《弱点》《大空头》《高频交易员》《第五风险》《掉队》等著作、Bloomberg专栏、Apple播客"Against the Rules"和公开访谈，提炼为可运行的分析框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/michael-lewis-skill
```

或复制到 `~/.config/opencode/skills/michael-lewis-skill/`。

## 使用示例

### 问：为什么我觉得市场不公平？

回答会用"信息不对称定位器"分析你所在的交易场景——谁比谁多知道什么。然后用"系统失灵诊断"三步法（激励→信息→问责）拆解不公平的根本原因，最后给出你是该"换位置"还是"抹平信息差"。

### 问：一个冷门行业怎么找机会？

回答会用"反直觉探测器"识别行业共识中可能错误的假设，再用"弱势逆袭公式"帮你找到资源有限时可以利用的系统盲点。最后用"叙事炼金术"把你的发现包装成可传播的洞见。

### 问：我有个重要判断，怎么验证它是不是对的？

回答会用"智力诚实检查"清单逐一过一遍——特别关注"你需要看到什么才会改变主意"这个核心问题，帮你区分"洞见"和"自我说服"。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"金融写作方法论"不同，这套框架的出发点是：

1. **从底层逻辑出发**：不教你怎么写故事，教你"信息不对称在哪里、系统漏洞在哪里"，你拿到这些判断标准，自己就能分析任何行业。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和刘易斯写作的固有偏见，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
