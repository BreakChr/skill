# mj-demarco-skill

> *"To make millions, you must impact millions."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**MJ·德马科的财富快车道思维框架。用三条道路（人行道/慢车道/快车道）拆解财富路径，用NECST五大戒律评估商业模式。**

基于《百万富翁快车道》《Unscripted》两部著作、Fastlane Forum社区、播客访谈和批评者分析，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/mj-demarco-skill
```

或复制到 `~/.config/opencode/skills/mj-demarco-skill/`。

## 使用示例

### 问：我的SaaS月收入5万，但感觉不像被动收入

回答会先让你过NECST五关——检查你是不是真的做到了Scale和Time。如果你的收入高度依赖你的个人参与，框架会告诉你："你不是在做生意，你是在做一份月薪5万的S象限工作。"

### 问：存了50万，该买房还是创业

回答会用"道路识别器"定位你现在在慢车道，用WCCA分析最坏结果的承受能力，然后用"财富公式"比较两条路径的上限——帮你做出选择。

### 问：想创业但没方向

回答会从"效应法则"出发：别问"我该做什么产品"，问"市场上有什么问题"。你的商业机会不在脑子里，在你每天抱怨的日常里。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"创业建议合集"不同，这套框架的出发点是：

1. **数学公式化**：不教你"怎么成功"，给你财富公式、NECST检查表、WCCA分析——你拿框架自己就能判断。

2. **从市场需求出发**：任何想法的起点不是"我有什么技能"，而是"市场有什么问题"。

3. **系统优先**：框架的核心判断标准是"你不在的时候生意转不转"——这是快车道和工作的分界线。

4. **诚实面对局限**：公开标注作者偏见和框架的适用边界，避免盲目套用。

5. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
