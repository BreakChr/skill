# 决策模式与行为分析

## 德马科的决策模式

### 模式1：首要问题导向

德马科的启动逻辑和大多数人相反：

**普通人**：我有什么资源？我有什么技能？我能做什么？→ 找市场
**德马科**：市场有什么问题？多大？谁在为此付费？→ 配资源

这个翻转的本质是从"供给导向"到"需求导向"——不是问你有什么，而是问市场缺什么。

### 模式2：数学优先

面对任何商业机会，德马科会先算：

```
市场规模 × 转化率 × 客单价 × 利润率 = ?
```

如果算出来的数字到不了七位数（100万+），不考虑。不是傲慢——而是在他的框架里，这就是快车道的入场券。

### 模式3：系统化本能

德马科看任何问题都会问"能不能做成系统"：

- 你做了一次的 → 能不能标准化复制？
- 你亲自做的 → 能不能教给别人做？
- 你盯着的 → 能不能自动化？

这不是"偷懒"——是把你的时间从执行中解放出来去思考更大的问题。

### 模式4：最小可行性，但方向要对

德马科和精益创业的区别：
- 精益创业：快速试错后再调整方向
- 德马科：先确定NECST五条都满足，再试

不是先做再看，是**先分析确定方向是对的，再低成本试错**。"试错"测试的是执行细节，不是方向本身。

## 常见误用

这套框架最常见的三种错误运用：

1. **把"快车道"当速成借口**：以为快车道就是"不用努力就能快速致富"。实际上快车道前期的努力远超慢车道——只是回报曲线不同。

2. **把"反慢车道"当反储蓄**：德马科反对的是"只靠存钱+指数基金"的单一策略。不是让你不储蓄——是让你在工作之外同时建系统。

3. **照搬NECST而不考虑阶段**：初创企业在"Entry"上可能天然低——这是正常的。NECST是评估最终状态的工具，不是要求你第一天就满足所有条件。

## 内在张力

- **个人奋斗 vs 结构优势**：德马科强调个人选择和执行力决定一切，但实际上起点、资源、信息差和运气对结果的权重远高于叙事中承认的。
- **反体制 vs 利用体制**：德马科雇用了律师、会计、公司结构、银行系统——他反对体制，但精于利用体制。对刚起步的人来说，这两者的界限很难把握。
- **确定性语气 vs 真实概率**：德马科用100%确定的口吻说"如果你做了X，就会得到Y"，但真实世界的商业成功有巨大的随机性。
