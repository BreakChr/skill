# nassim-nicholas-taleb-skill

> *"Antifragility is beyond resilience or robustness. The resilient resists shocks and stays the same; the antifragile gets better."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**纳西姆·尼古拉斯·塔勒布的不确定性思维框架。用黑天鹅、反脆弱性与非对称风险的底层逻辑拆解决策、风险管理和系统设计。**

基于Incerto系列五卷、学术论文、播客访谈和社交媒体持续输出，提炼为可运行的思维框架。

## 一键安装

```bash
npx skills add <你的GitHub用户名>/nassim-nicholas-taleb-skill
```

或复制到 `~/.config/opencode/skills/nassim-nicholas-taleb-skill/`。

## 使用示例

### 问：创业到底该All-in还是留后路？

回答会用"杠铃策略"来拆解——不选中间路线，一端保底线（稳定收入），一端赌上限（高风险项目），放弃中等风险的中庸方案。

### 问：基金经理推的量化模型靠谱吗？

回答会用"切身利害检验器"来拆解——基金经理是否把自己的钱投在同一个策略里？如果没有，他的建议权重为零，不管他的学历和title有多高。

### 问：现在该不该卖房炒股？

回答会用"三元结构分类器"——先判断你的资产结构是脆弱、强韧还是反脆弱。然后用"否定法"反向排错，找出最坏情况下能否承受，再决定。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"不确定性与风险"类工具书不同，这套框架的出发点是：

1. **从底层逻辑出发**：不教你如何预测未来，教你"如何在不预测的情况下做决策"。你拿到的是分类和过滤工具，不是"在这个行情下怎么做"的临时建议。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
