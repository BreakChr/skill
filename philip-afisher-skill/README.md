# philip-a-fisher-skill

> *"The time to sell a stock is almost never, if the job has been correctly done when it was purchased."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)(LICENSE)

**菲利普·费雪的成长投资思维框架。用企业质量、管理层能力和竞争优势的底层逻辑拆解投资决策。**

基于《怎样选择成长股》（Common Stocks and Uncommon Profits）、《保守型投资者夜夜安寝》等经典，及70年华尔街实践，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/philip-a-fisher-skill
```

或复制到 `~/.config/opencode/skills/philip-a-fisher-skill/`。

## 使用示例

### 问：这家公司市盈率很高，是不是太贵了？

回答会用"逆向钟摆"模型分析——好公司暂时的高PE可能正是买入时机，关键看15点清单是否仍然通过。费雪关心的不是"贵不贵"，而是"够不够好"。

### 问：我想分散投资，买20只不同的股票

回答会用"集中度检测仪"拆解——真正了解20家公司几乎不可能，6-12只经过深度研究的标的比20只蜻蜓点水的持仓更安全。

### 问：这只股票涨了50%，该卖了吗？

回答会用"买入并持有计算器"分析——卖出好公司的理由不应是"它涨了"。只要基本面没变坏、竞争优势没消失，持有才是最优策略。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

1. **从定性分析出发**：不教你看K线和技术指标，教你"如何判断企业质量和管理层能力"，你拿到这个标准，自己就能评估任何公司。

2. **可操作的检查清单**：每个思维模型附带具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
