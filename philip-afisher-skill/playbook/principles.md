# 核心原理详解

## 投资的第一性原理

费雪体系最底层的一个判断：**投资的核心不是预测价格，而是判断企业质量。**

- 好企业 → 长期增长 → 股价跟随基本面上升
- 差企业 → 长期萎缩 → 任何择时都无法挽救
- 你真正理解一家企业 → 你才有持有它的信心

所有复杂投资策略、估值模型、技术分析，拆到底层都是这条。所以费雪反复强调"定性分析优先"——不是因为定量不重要，而是数字说明过去，人决定未来。

## 闲聊法（Scuttlebutt）的本质

闲聊法不是八卦，不是小道消息，而是一种**系统性获取非公开信息**的方法。

原理：
- 公开信息（财报、公告、研报）所有人都能看到，没有信息优势
- 非公开信息（客户体验、供应商态度、员工士气、竞争对手的恐惧）才是真正的认知差
- 通过交叉验证多个来源的碎片信息，你构建的是一个比华尔街更立体的图景

约束条件：
- 这是最累的方法——需要走出去和人聊，不是坐在电脑前看数据
- 这是最慢的方法——一次完整的闲聊调研可能耗时数月
- 但也是最有效的方法——因为大多数人不愿意做

## 关于成长与价格的矛盾

费雪对"成长vs价格"的态度是这套框架中最与格雷厄姆式价值投资不同的部分：

- 格雷厄姆：找到足够便宜的东西，安全边际来自价格远低于内在价值
- 费雪：找到足够好的东西，只要不是贵得离谱，安全边际来自企业持续增长的能力
- 巴菲特：85%格雷厄姆 + 15%费雪 = 以合理价格买入优质企业

费雪不是不在乎价格，而是认为"好企业的增长最终会消化任何看似合理的估值"。

## 持有时间观念

费雪的持有周期有一种极端的耐心：

- 短期：不关注季度波动，不参与市场情绪
- 长期：真正的大回报来自持有10-30年
- 核心信念：优秀企业会持续复利，你的干扰（交易）才是最大的成本

这不是被动投资——这是一种"在研究上极度活跃，在交易上极度被动"的节奏。花数月研究，花数十年持有。
