# robert-hfrank-skill

> *"The cost-benefit principle says that an action should be taken only if its additional benefit exceeds its additional cost."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**罗伯特·弗兰克的行为经济学思维框架。用成本收益、赢者通吃、地位军备竞赛三大核心理论拆解消费、市场与公共政策。**

基于《牛奶可乐经济学》《赢者通吃的社会》《成功与运气》《奢侈病》《达尔文经济学》等著作、纽约时报专栏及40年学术研究，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/robert-hfrank-skill
```

或复制到 `~/.config/opencode/skills/robert-hfrank-skill/`。

## 使用示例

### 问：我每年赚200万但还是觉得不够花

回答会用"地位军备竞赛"和"支出瀑布"框架诊断：你的支出增长不是因为你需要更多，而是因为你所处的参照群体变了。然后给"位置性vs非位置性消费"的分拆工具，帮你识别哪些支出是在买相对位置而非生活质量。

### 问：AI会让所有人都失业吗

回答会用"赢者通吃市场"框架分析：AI加速了赢者通吃，但不是消灭所有工作——它让顶层少数人获得指数级回报，中层的标准化工作被压缩，底层服务类工作反而更稳定。关键问题不是"工作还在不在"，而是"回报分配怎么变"。

### 问：为什么我朋友创业比我努力但赚得没我多

回答会用"成功运气分解器"拆解：在赢者通吃市场中，微小初始优势在不断放大。然后引导用户做"运气剔除"——如果把随机因素去掉，你们的真实能力差距有多大？同时指出"运气盲点"让你高估了自己的策略选择，低估了朋友的时运不济。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

1. **从日常现象出发**：不教数学公式和供需曲线，从"为什么牛奶盒是方的"这种问题走进经济学
2. **可操作的判断工具**：每个思维模型附带具体的"用法"说明
3. **诚实面对局限**：公开标注作者立场倾向和框架适用边界
4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用
