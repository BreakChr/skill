# 核心原理详解

## 市场非有效性的第一性原理

席勒体系最底层的一个判断：**资产价格的波动幅度远大于基本面变化所能解释的幅度。**

- 如果市场完全有效 → 价格变化只能由新信息驱动 → 波动幅度应等于信息的不确定性
- 席勒的实证（1981年经典论文）→ 股票价格波动是股息波动的5-13倍
- 结论 → 心理和叙事因素主导短期和中期定价

这个发现动摇了有效市场假说的根基。法玛说市场是有效的，席勒说你有数据吗？法玛说有理论，席勒说我有证据。

## CAPE的时间哲学

CAPE不是普通的估值指标，它体现了一种时间观：

- **标准PE**：看过去12个月的盈利 → 被经济周期严重扭曲（衰退期PE虚高，繁荣期PE虚低）
- **CAPE**：看过去10年通胀调整后的平均盈利 → 穿越周期，看到企业的真实盈利能力

核心洞察：一个企业的盈利在一年内可能波动50%以上，但十年的平均盈利会平滑到接近真实生产力水平。用十年的数据看估值，是对市场短视的一种对抗。

## 叙事病毒的生命周期

席勒的叙事经济学把经济叙事比作病毒，有其生命周期：

1. **变异** — 一个想法在社会中产生，适应了当时的焦虑或欲望
2. **传播** — 被媒体、意见领袖、社交网络放大
3. **感染** — 大量人接受这个叙事并据此行动
4. **饱和** — 几乎每个人都听说过，增长放缓
5. **衰退** — 被新叙事取代，或与事实矛盾导致失效
6. **潜伏** — 可能在未来条件适当时复发（如"通胀暂时论"反复出现）

## 动物精神的层次

席勒和阿克尔洛夫把动物精神分为五个层次，从个体到集体：

- **个人层面**：情绪影响每个人的经济决策
- **人际层面**：信任和不信任决定交易成本
- **市场层面**：羊群行为制造趋势和逆转
- **制度层面**：公平感决定人们对税收、监管的态度
- **叙事层面**：故事连接所有层次，形成集体行动

## 金融民主化的矛盾

席勒既是金融创新的倡导者，又是华尔街的批评者：

- 批评：现有的金融体系太复杂、太不透明、只服务富人
- 倡导：用金融工具分散每个人的生活风险，而非只用于投机
- 具体方案：GDP挂钩债券、职业保险、住房权益保险
- 困境：好的金融创新往往被坏的金融创新淹没，因为后者短期利润更高
