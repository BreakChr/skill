# robert-toru-kiyosaki-skill

> *"An asset puts money in your pocket. A liability takes money out."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**罗伯特·清崎的财富思维框架。用资产vs负债的底层逻辑拆解产品、商业和财务决策。**

基于《富爸爸穷爸爸》系列、《Cashflow现金流》游戏、30+播客访谈、社交媒体持续输出和批评者分析，提炼为可运行的思维框架。


## 使用示例

### 问：我的SaaS月收入5万，但感觉像在打工

回答会先反问"你在经营生意还是经营工作？"，然后用ESBI象限定位你当前在S还是B，再给出从S到B的迁移路径——建一个不依赖你的系统。

### 问：AI时代什么工作不会被替代

回答会用杠杆思维拆解：AI替代E象限的按指令做事的人，放大B象限的建系统者和I象限的资本配置者。然后用"反问开关"把用户的问题从"找一份AI替代不了的工作"重新定义为"怎么用AI做我的杠杆"。

### 问：想创业但觉得市场太饱和了

回答会从"四象限定位器"出发，分析你是想做B还是做S——大多数人以为自己在"创业"，实际上是在"给自己找了一份没有底薪的工作"。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"财商建议合集"不同，这套框架的出发点是：

1. **从底层逻辑出发**：不教你怎么选股票买基金，教你"资产和负债的本质区别是什么"，你拿到这个判断标准，自己就能分析任何项目。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界和争议点，避免盲目套用。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
