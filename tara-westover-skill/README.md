# tara-westover-skill

> *"Education is not about becoming someone new. It's about becoming who you already are."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**塔拉·韦斯特弗的教育觉醒与自我重塑思维框架。用教育vs规训的底层逻辑拆解认知边界、身份冲突和人生选择。**

基于《你当像鸟飞往你的山》回忆录、TED演讲、30+媒体访谈和批评者分析，提炼为可运行的思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/tara-westover-skill
```

或复制到 `~/.config/opencode/skills/tara-westover-skill/`。

## 使用示例

### 问：我该不该离开父母去外地发展？

回答会先帮你定位冲突类型——是忠诚冲突还是价值冲突？然后用"认知牢笼检测器"看你目前的环境是否在控制你的信息接入。最后用"飞往你的山"矩阵评估你的准备度，而不是直接说"去吧"或"留下"。

### 问：读了大学后发现和家里越来越没话说了

回答会用"身份冲突分析仪"拆解你的矛盾——你不是变坏了，你是在做两个版本的自己。然后用"知识·权力透镜"帮你看到教育在你和家庭之间制造的张力不是你的错。

### 问：感觉被工作/关系困住了，但不确定是不是自己太矫情

回答会用"教育vs规训二分器"帮你看到你身处的环境是在拓展你还是限制你。再用"记忆棱镜"让你审视你对困境的叙述本身——你的困住感来自事实还是来自你对自己说的故事？

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"励志成长建议"不同，这套框架的出发点是：

1. **从认知结构出发**：不教你怎么"和解"或"放下"，教你"认知牢笼是怎么建起来的"，你拿到这个分析工具，自己就能诊断任何困境。

2. **可操作的判断工具**：每个思维模型附带一个具体的"用法"说明——读完就知道怎么用。

3. **诚实面对代价**：公开标注"自我重塑"的代价，不贩卖"做自己"的廉价感动。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
