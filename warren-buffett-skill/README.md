# warren-buffett-skill

> *"It's far better to buy a wonderful company at a fair price than a fair company at a wonderful price."*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**沃伦·巴菲特的价值投资思维框架。用内在价值、护城河、能力圈的底层逻辑评估企业质量与投资决策。**

基于70年致股东信（1956-2026）、伯克希尔年会实录、查理·芒格合伙时期的框架进化、以及公开访谈，提炼为可运行的价值投资思维框架。

---

## 一键安装

```bash
npx skills add <你的GitHub用户名>/warren-buffett-skill
```

或复制到 `~/.config/opencode/skills/warren-buffett-skill/`。

## 使用示例

### 问：XX股票现在跌了很多，可以买吗？

回答不会直接说"买"或"不买"，而是用**内在价值标尺**和**安全边际**框架引导你先估算企业的真实价值，再对比当前价格判断折扣是否足够大。同时用**能力圈边界仪**让你诚实地评估自己是否真的懂这个生意。

### 问：AI时代来了，价值投资还管用吗？

回答用**护城河识别器**拆解：技术变革确实会侵蚀一些企业的护城河（柯达、诺基亚），但也会强化另一些（软件企业的高转换成本、网络效应平台）。核心不是"不碰科技"，而是"不碰你不懂的生意"——巴菲特对苹果的投资就是最好的例子。

### 问：我该分散投资还是集中投资？

回答从**集中与耐心**模型出发解释：如果你真的理解一个生意，就不需要分散50个来保护无知。但前提是你必须诚实地面对自己的认知边界。当你说不清楚第5个最好的想法时，集中在前3个就足够了。

更多场景在 [cases/scenarios.md](cases/scenarios.md)。

## 设计思路

和市面上多数"投资建议合集"不同，这套框架的出发点是：

1. **从底层逻辑出发**：不推荐任何具体股票或行业，教你"如何判断一个生意是否值得长期拥有"，你拿到这个框架，自己就能分析任何企业。

2. **可操作的判断工具**：每个思维模型附带具体的"用法"和"自测方法"——读完就知道怎么用。

3. **诚实面对局限**：公开标注模型的适用边界——巴菲特的规模优势普通人没有，集中回撤的心理考验不是每个人都能承受。

4. **轻量**：不依赖任何外部文件或API，拷走SKILL.md就能用。
